df_mclt=pd.read_sql("""
                          SELECT m.country_id,c.name AS country_name,m.league_id,l.name AS league_name,m.season,m.stage,m.date
                                ,m.home_team_api_id,m.away_team_api_id,m.home_team_goal,m.away_team_goal,t.team_short_name AS home_team_name
                          FROM Match m 
                          JOIN Country c
                          ON m.country_id=c.id
                          JOIN League l
                          ON m.league_id=l.id
                          JOIN Team t
                          ON t.team_api_id=m.home_team_api_id;

研究问题 3¶

哪支球队赢得胜利的次数最多？

西班牙的皇家马德里Real Madrid CF胜利次数最多，获胜次数达141次。

FIFA国际足联战术设计的三个基本部分buildUpPlay、chanceCreation、defence，决定了一支球队的进攻和防守方式。

BuildUpPlaySpeed抢下球反击的推进速度，Real Madrid CF平均50.666667。所有队平均52.463967
ChanceCreationPassing传球，Real Madrid CF平均67.5。所有队平均52.166781
ChanceCreationShooting射门，Real Madrid CF平均69。所有队平均53.971860
DefencePressure防守紧逼，Real Madrid CF平均52。所有队平均46.017845

反击稳，射门准，防守狠
